Про застосунок (більш додатково в файлі README.md)

Це веб-застосунок, що дозволяє користувачеві ввести будь-яку суму та набір номіналів монет, після чого знаходить мінімальну кількість монет, необхідних для утворення цієї суми.

Як працює:

  1. Користувач вводить суму (наприклад, 67)
  2. Вводить номінали монет (наприклад, 1,5,10,25)
  3. Натискає кнопку “Обчислити”
  4. Алгоритм знаходить оптимальний розмін монет (жадібний підхід)
  5. Результат виводиться у вигляді списку: скільки яких монет потрібно
